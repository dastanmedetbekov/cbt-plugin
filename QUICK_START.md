# Quick Start

## Install in 3 Steps

1. **Copy files to your Obsidian vault:**
   ```bash
   mkdir -p /path/to/your/vault/.obsidian/plugins/cbt-journal
   cp main.js manifest.json styles.css /path/to/your/vault/.obsidian/plugins/cbt-journal/
   ```

2. **Enable the plugin:**
   - Open Obsidian → Settings → Community Plugins
   - Turn OFF "Safe mode"
   - Enable "CBT Journal"

3. **Start journaling:**
   - Click the brain icon in the left sidebar
   - Fill out the form and click "Save Entry"

## What You'll Get

Every entry includes:
- Structured thought analysis
- Emotion tracking (0-10 scale)
- 10 cognitive distortions to identify
- Space for rational alternative thoughts
- Automatic markdown notes with tags
- Beautiful, organized entries

## Example Workflow

1. Something triggers negative thoughts
2. Open CBT Journal (brain icon)
3. Describe the situation
4. Write your automatic thoughts
5. Rate your emotions (0-10)
6. Identify cognitive distortions
7. Write a more rational perspective
8. Rate emotions again - see the improvement!
